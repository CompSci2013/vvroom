## Pop-Out Window Styling

Pop-out windows require special handling to hide scrollbars:

```scss
body.popout-body {
  overflow: hidden;
  height: 100vh;
  width: 100vw;
}

html.popout-html {
  overflow: hidden;
  height: 100vh;
  width: 100vw;

  .app-content {
    overflow: hidden !important;
    height: 100vh !important;
  }
}
```
