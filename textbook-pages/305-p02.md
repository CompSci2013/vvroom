### Validation vs. Sanitization

The validator does two things:

1. **Validate** — Check for errors, return result object
2. **Validate and Sanitize** — Check for errors, apply defaults, return clean config

```typescript
// Validation only
const result = validator.validate(config);
if (!result.valid) {
  console.error(result.errors);
}

// Validation with sanitization (applies defaults)
const cleanConfig = validator.validateAndSanitize(config);
// Throws if invalid, returns sanitized config if valid
```
