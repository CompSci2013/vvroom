### Example Request

```
GET /vehicles/details?manufacturer=Toyota&yearMin=2020&yearMax=2024&page=1&size=20&sortBy=year&sortOrder=desc
```

### Response Shape

```typescript
{
  "results": VehicleResult[],
  "total": number,
  "page": number,
  "size": number,
  "totalPages": number,
  "statistics": VehicleStatistics
}
```
